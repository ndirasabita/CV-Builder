## Tentang Proyek

CV-Builder dibuat sebagai solusi cepat dan mudah bagi siapa saja yang butuh CV profesional tanpa perlu software berat â€” cukup browser.  
Dengan antarmuka bersih dan template ramah ATS, pengguna dapat:  
- Mengisi data pribadi, riwayat pendidikan / pekerjaan, skill, portofolio, dll.  
- Mendapat preview langsung di browser.  
- Mengunduh CV dalam PDF siap pakai.  

Proyek ini berguna bagi mahasiswa, fresh graduate, atau siapa pun yang ingin membuat CV dengan mudah dan cepat.  

## Fitur

- Form input dinamis (tambah / hapus entri pendidikan, pengalaman, skill, dsb.)  
- Preview CV realtime
